Demo do Widget de Chat

Este é um exemplo de página que hospeda o widget de chat. O widget aparecerá flutuando no canto inferior direito.

Como usar em produção:

<script src="https://seu-dominio.com/chatbot.js"></script>
<script>
  ChatBot.init({
    webhookUrl: 'https://seu-n8n.com/webhook/chat',
    title: 'Atendimento',
    welcomeMessage: 'Olá! Como posso ajudar?',
    primaryColor: '#3B82F6'
  });
</script>

Nota: Em modo de desenvolvimento, o widget é renderizado diretamente no React. Após o build, ele funcionará como um script standalone que pode ser incluído em qualquer site.
